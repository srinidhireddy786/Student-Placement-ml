Student Placement & Salary Predictor

An end-to-end Machine Learning web application that predicts whether a student will be placed and estimates their expected salary (in LPA) based on academic and skill-based features.

🚀 Features

🎯 Placement Prediction (Classification)

💰 Salary Prediction (LPA) (Regression)

📊 Feature Importance Visualization

📈 CGPA vs Placement Probability Graph

🧠 Inputs include:

CGPA

Internships

Projects

Certifications

DSA Score

Aptitude Score

Branch (CSE, ECE, etc.)

College Tier

Gender
